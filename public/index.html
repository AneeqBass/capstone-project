USER INFORMATION
HTTP Method Path Description
GET /users Gets all users
GET /users/:id Gets a single user by id
POST /users/register Adds a new user
PATCH /users/update/:id Edit a user's information by id
LOGIN /users/login User login
DELETE /users/delete/:id Removes a user by id
